# net-reverse-page

Net Reverse — AI automation solutions for business.

## Tech stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- React Router v6 (i18n: en / ru / uk)
- Deployed on Railway

## Development

```sh
bun install
bun run dev
```

## Build

```sh
bun run build
```

## Deploy

Deployed via Railway. Push to `main` triggers a new deploy.
